# Software Specification Document: "Getting Things Done" Game

## 1. Executive Summary

### 1.1 Overview
"Getting Things Done" is a time management simulation game where players assume the role of a teacher managing a typical school week. The game will be implemented as a Progressive Web App (PWA) using HTML5 and JavaScript, deployable via a single URL and optimized for mobile devices.

### 1.2 Core Objectives
- Simulate 40 rounds (5 days Ã— 8 rounds/day) of teacher workload management
- Balance teaching, administrative tasks, collaborative activities, and personal energy
- Provide engaging visual feedback reflecting player state and decisions
- Enable easy customization of game parameters, rules, and visual assets

## 2. Technical Architecture

### 2.1 Technology Stack
- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Architecture**: Single Page Application (SPA)
- **Deployment**: Static hosting (single HTML file with inline/minimal external assets)
- **Progressive Web App**: Service Worker for offline capability, manifest.json for installability

### 3.2 Rule Engine Configuration

```javascript
const GAME_RULES = [
  // Rule 1: Staff Meeting Required
  {
    name: 'staff_meeting_scheduled',
    condition: (state) => {
      return state.fixedEvents.includes(state.currentRoundId);
    },
    action: (state) => {
      state.availableActions = ['attend_meeting', 'skip_meeting'];
      state.mandatoryAction = 'attend_meeting';
    }
  },
  
  // Rule 2: Disruption Occurs
  {
    name: 'disruption_active',
    condition: (state) => {
      return state.currentDisruption !== null;
    },
    action: (state) => {
      state.actionSuccess = false;
      state.disruptionMessage = getDisruptionMessage(state.currentDisruption);
    }
  },
  
  // Rule 3: Energy Depleted
  {
    name: 'energy_depleted',
    condition: (state) => {
      return state.energy <= 0;
    },
    action: (state) => {
      state.availableActions = ['break'];
      state.forceBreak = true;
    }
  },
  
  // Rule 4: End of Day - Admin Compounding
  {
    name: 'end_of_day_admin',
    condition: (state) => {
      return state.isEndOfDay && state.adminRemaining > 0;
    },
    action: (state) => {
      state.adminRemaining = state.adminRemaining * GAME_CONFIG.adminCompounding.multiplier;
      if (state.adminRemaining > GAME_CONFIG.penalties.adminMaxEndOfDay) {
        state.penalties.admin += 1;
      }
    }
  },
  
  // Rule 5: End of Day - Teaching Minimum
  {
    name: 'end_of_day_teaching',
    condition: (state) => {
      return state.isEndOfDay && state.dailyTeachingCompleted < GAME_CONFIG.penalties.teachingMinPerDay;
    },
    action: (state) => {
      state.penalties.teaching += 1;
    }
  },
  
  // Rule 6: End of Game - Collaborative Minimum
  {
    name: 'end_game_collaborative',
    condition: (state) => {
      return state.isGameEnd && state.collaborativeCompleted < GAME_CONFIG.penalties.collaborativeMinTotal;
    },
    action: (state) => {
      state.penalties.collaborative += 1;
    }
  }
];
```

### 5.2 State Transitions

```javascript
class GameEngine {
  // Phase 1: Action Selection
  startSelectionPhase(state) {
    state.phase = 'selection';
    state.availableActions = this.calculateAvailableActions(state);
    this.applyRules(state, 'selection');
  }
  
  // Phase 2: Action Execution
  executeAction(state, action) {
    state.selectedAction = action;
    state.phase = 'result';
    
    // Check for disruption
    if (this.isDisruptionRound(state)) {
      state.actionSuccess = false;
      state.currentDisruption = this.getRandomDisruption();
    } else {
      state.actionSuccess = true;
      this.processAction(state, action);
    }
    
    this.applyRules(state, 'result');
  }
  
  // Advance to next round
  advanceRound(state) {
    state.actionHistory.push({
      round: state.currentRoundId,
      action: state.selectedAction,
      success: state.actionSuccess
    });
    
    state.currentRoundId++;
    state.currentRound++;
    
    // Day transition
    if (state.currentRound > 8) {
      this.endDay(state);
      state.currentDay++;
      state.currentRound = 1;
      this.startDay(state);
    }
    
    // Game end
    if (state.currentRoundId > 40) {
      this.endGame(state);
    }
  }
}
```

## 6. Rule Engine Implementation

### 6.1 Rule Evaluation System

```javascript
class RuleEngine {
  constructor(rules) {
    this.rules = rules;
  }
  
  evaluate(state, phase = null) {
    for (const rule of this.rules) {
      // Filter by phase if specified
      if (rule.phase && rule.phase !== phase) continue;
      
      // Evaluate condition
      if (rule.condition(state)) {
        // Execute action
        rule.action(state);
        
        // If rule is terminal, stop evaluation
        if (rule.terminal) break;
      }
    }
  }
  
  addRule(rule) {
    this.rules.push(rule);
  }
  
  removeRule(ruleName) {
    this.rules = this.rules.filter(r => r.name !== ruleName);
  }
}
```

- **Font Size**: Minimum 16px for body text
- **Color Independence**: Information not conveyed by color alone
- **Touch Gestures**: No complex gestures required
